User Instructions:


The simulator consists of three regions, Variable window, Simulation control buttons and Result display window.

 

Variable Window:

 

Student can enter the values for the variables in the model. Parameters include (hw) Handling of worm, (aw) Abundance of worm, (hb) Handling of beetle, (ab) Abundance of beetle




Simulation Control Buttons:


Simulation control includes: Run, by pressing this button simulation will perform and Reset, by pressing this button the whole simulator will get reset.



Result Display Window:


The simulated values for the parameters are interspersed as graphs in this window.



Step by Step Explanation:


Step 1: Open the simulator and enter the values for the parameters in the variable window.


Step 2: Click Run button to simulate the model


 

The graph will be plotted in the Result display window.


Step 4: Click Reset button to reset the simulator.
